변수란 프로그램 작업을 처리하기 위해 하나의 값을 저장할 수 있는 메모리 공간이다.
자바에서는 다양한 타입을 저장할 수 없고, 한가지 타입만 값으로 저장될 수 있다.
변수 선언 위치는 제약 없다. 어디에서나 변수의 선언이 가능하나 되도록 맨 위

논리형 boolean
문자형 char(문자열: String)
정수형 byte, short, int, long
실수형 float, double
각 형식에 해당 byte에 맞는 값을 넣는다.

기본 데이터 타입
-메모리에 있는 실제값 = 변수 데이터 값
-주로 간단한 데이터(예:int double, boolean, char 등..)

참조 데이터 타입
-메모리에 있는 실제값 = 변수가 저장된 주소(실제 변수 데이터는 heap영역
-여러 가지 데이터들이 모여 있는 복잡한 데이터로 기본 자료형에 비해 크기가 크다
(예 : String, System 등의 객체, 배열 등등)
