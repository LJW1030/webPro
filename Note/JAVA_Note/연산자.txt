연산자
: 일정한 규칙을 가지고 계산하는 것

① 산 술  연 산 : *  /  +  -  %
② 증감연산 : ++  --
③ 동등관계(비교)연산 : ==(같다)  !=(다르다)  <  <=  >  >=
④ 논 리  연 산 : &&  || &  |  !
⑤ 삼항(3항)연산: ? :
⑥ 할당(대입)  연산자 : =  +=  -=  *=  /=  %=

&&연산자의 좌항이 false일 경우 우항 실행 안 함
||연산자의 좌항이 true일 경우 우항 실행 안 함

삼항연산자
형식 -> 조건식 ? 참일경우의 값/식 : 거짓일 경우의 값/식
조건식 : 식의 결과가 참/거짓으로 나눠지는 식

비트연산자
비트를 사용해 연산하는 연산자
정수와 정수 형태의 연산만 가능
&(and), |(or), ^(Xor)
